# ♻️ AI-Powered Smart Waste Intelligence Platform

An interactive AI platform that combines deep learning, computer vision, explainable AI, and sustainability analytics to classify recyclable waste.

---

## Features

- ♻️ AI-powered waste classification
- 📊 Sustainability analytics dashboard
- 🔥 Grad-CAM explainability
- 🤖 Multi-model benchmarking
- 🌍 Environmental impact estimation
- 📈 AI forecasting
- 📷 Webcam inference
- 📄 PDF report generation

---
